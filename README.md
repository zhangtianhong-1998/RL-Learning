# Mathematical Foundations of Reinforcement Learning (中文翻译版)

本仓库提供西湖大学 WindyLab 实验室 **Shiyu Zhao 老师** 所著《Mathematical Foundations of Reinforcement Learning》的中文翻译版本。

> [!NOTE]
> 本项目主要用于个人学习交流。基于 **Gemini 3 Pro** 模型辅助翻译，并结合个人理解在飞书文档中进行了重组与校对。

---

## 📖 项目简介

本项目旨在为强化学习初学者和研究者提供一份高质量的中对照参考资料。原著以数学严谨性著称，深入浅出地讲解了强化学习的核心理论。

* **原著地址**：[MathFoundationRL (GitHub)](https://github.com/MathFoundationRL)
* **原著视频**：[Bilibili](https://www.bilibili.com/video/BV1Dg41127zU/)
* **翻译工具**：Gemini 3 Pro (截图识别 + 全文翻译)
* **整理平台**：飞书文档 (Lark Doc)
* **包含内容**：正文翻译、数学公式重排、个人理解注释。

## ⚠️ 注意事项

1. **翻译局限性**：由于涉及大量数学推导和专业术语，AI 翻译可能存在不准确之处，部分公式格式在转换过程中可能偶有错位。
2. **个人注释**：文档中包含部分译者的理解性笔记，阅读时请注意区分原著内容与个人观点。
3. **反馈建议**：如果你在阅读过程中发现术语翻译不当、公式错误或链接失效，欢迎提交 **Issue** 帮助改进。
